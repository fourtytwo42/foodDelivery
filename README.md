# Food Delivery Application

A food delivery application built with Node.js.

## Prerequisites

- Node.js (v20+)
- npm
- PostgreSQL
- PM2 (for process management)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/fourtytwo42/foodDelivery.git
cd foodDelivery
```

2. Install dependencies:
```bash
npm install
```

3. Set up the database:
```bash
# PostgreSQL should be running
# Create your database and configure connection
```

## Running the Application

### Development
```bash
npm start
```

### Production (using PM2)
```bash
pm2 start app.js
```

- Node.js
